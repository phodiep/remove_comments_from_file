# remove_comments_from_file

Given a file, remove all comments from the file.

Single line comments
>  // this is a comment \n
  
Multiline comments
  
>  `/* this is a comment`

>   ` * still part of comment`

>   ` * still commenting // <-this is still part of comment`

>   ` * another line /* of comment`

>   ` */`
